# System Info App

# Requisitos funcionais

- O app deve ser um tray
- O app não deve ter bordas e ser translucido/opaco
- O app deve ter um menu com as seguintes opções:
  - Mostrar
  - configurações
  - Sair
- quando o app esta com a janela aberta deve mostrar as informações do computador
    - Uso da CPU
    - Uso de memoria
    - Uso de disco
    - Espaço em disco
    - Uso GPU (opcional)
  - as configurações:
    - aumentar tamanho da fonte
    - mudar cor do texto
    - trocar fonte

# planejamento
